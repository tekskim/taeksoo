Cluster 상세 페이지 — 상태별 안내 메시지

Provisioning Cluster provisioning is in progress. This typically takes a few minutes. Status will become 'Provisioned' when provisioning is complete.
Deleting Cluster deletion is in progress. All resources will be cleaned up automatically.
Processing A cluster operation is in progress. Some actions may be temporarily unavailable. Status will return to 'Provisioned' once the operation is complete.
Failed Cluster provisioning failed at Control plane initializing.  View error logs in Logs
